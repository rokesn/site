🔓 우회 접속 방법 2026

방법 1 — DNS 서버 변경 (가장 쉬운 방법)

DNS를 변경하면 별도 프로그램 없이 차단된 사이트에 접속할 수 있습니다.

서비스기본 DNS보조 DNS
🌐 구글 DNS 8.8.8.8 8.8.4.4
☁ 클라우드플레어 1.1.1.1 1.0.0.1
  1. Windows 설정 열기시작 메뉴 → 설정 → 네트워크 및 인터넷
  2. 어댑터 옵션 변경이더넷 또는 Wi-Fi → 속성 클릭
  3. IPv4 속성 열기인터넷 프로토콜 버전 4(TCP/IPv4) 더블클릭
  4. DNS 주소 입력기본 DNS: 8.8.8.8 / 보조 DNS: 8.8.4.4 입력 후 확인
  5. 브라우저 재시작브라우저를 완전히 닫고 다시 열어 접속 시도

💡 이 가이드의 신뢰성

이 가이드는 여기여 운영팀이 직접 DNS 서버(8.8.8.8, 1.1.1.1)를 변경하며 테스트한 실제 방법을 기반으로 작성되었습니다. 저희는 매일 이 DNS 설정으로 여기여에 접속하고 있습니다.

운영자: 여기여 (Yugiyu) 운영팀 — 링크모음·주소모음 포털
테스트 환경: Windows 10/11, macOS, Android, iOS 전 기기에서 실제 테스트 완료
데이터 출처: Google Public DNS 공식 문서, Cloudflare 1.1.1.1 공식 문서 기준

방법 3 — VPN 사용

  1. 무료 VPN 앱 설치Proton VPN, Windscribe 등 신뢰할 수 있는 무료 VPN을 설치합니다.
  2. VPN 연결앱을 실행하고 해외 서버(일본, 미국 등)에 연결합니다.
  3. 여기여 접속VPN 연결 상태에서 https://yugiyu.com 으로 접속합니다.
💡 팁: 일상 접속은 DNS 변경을 추천합니다.
