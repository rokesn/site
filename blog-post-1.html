여기여 접속 안 될 때 해결 방법 — DNS 변경 완벽 가이드 2026

여기여 사이트에 갑자기 접속이 안 될 때 가장 빠르고 안전한 해결 방법은 DNS 서버를 변경하는 것입니다. 이 가이드에서는 구글 DNS(8.8.8.8)와 클라우드플레어 DNS(1.1.1.1)로 변경하는 방법을 Windows, Mac, Android, iOS 기기별로 단계별로 안내합니다.

여기여 접속이 안 되는 이유

여기여를 포함한 많은 링크모음 사이트들은 국내 인터넷 차단 정책(방통심의위원회의 불법·유해정보 차단)으로 인해 접속이 차단되는 경우가 있습니다. 이 차단은 ISP(KT, SK텔레콤, LG유플러스 등) 수준에서 DNS 차단 방식으로 이루어집니다.

DNS(Domain Name System)는 쉽게 말해 인터넷의 "전화번호부"입니다. 여러분이 주소창에 도메인 이름을 입력하면, DNS 서버가 해당 도메인의 IP 주소를 찾아줍니다. 국내 ISP들은 차단 대상 사이트의 도메인이 입력됐을 때 올바른 IP 주소 대신 차단 페이지를 반환합니다.

따라서 DNS 서버를 국내 ISP 기본값에서 구글이나 클라우드플레어의 공개 DNS로 변경하면, ISP의 차단 필터를 거치지 않고 직접 해당 사이트에 접근할 수 있습니다. 별도 프로그램 설치가 필요 없고, 무료이며, 속도 저하가 거의 없어 가장 추천하는 방법입니다.

추천 DNS 서버 목록

아래 두 가지 DNS 서버 중 하나를 선택해 사용하면 됩니다. 둘 다 무료이며 전 세계적으로 신뢰받는 서비스입니다.

서비스명기본(Primary) DNS보조(Secondary) DNS특징
구글 Public DNS8.8.8.88.8.4.4안정성 최고, 전 세계 1위
클라우드플레어 DNS1.1.1.11.0.0.1속도 최빠름, 개인정보 보호
💡 어떤 DNS가 더 좋나요? 두 DNS 모두 훌륭합니다. 클라우드플레어(1.1.1.1)가 일반적으로 응답 속도가 조금 더 빠르며, 구글(8.8.8.8)은 안정성이 검증된 서비스입니다. 둘 다 개인정보 정책이 명확하여 안전하게 사용할 수 있습니다.

Windows에서 DNS 변경 방법

Windows 10 / 11 공통

1설정 열기
키보드에서 Windows 키 + I를 눌러 설정을 엽니다. 또는 시작 버튼 → 설정(⚙️) 클릭.
2네트워크 및 인터넷 접속
네트워크 및 인터넷 항목을 클릭합니다. 현재 사용 중인 연결(Wi-Fi 또는 이더넷)을 선택합니다.
3어댑터 옵션 변경
페이지 하단 또는 오른쪽에서 어댑터 옵션 변경을 클릭합니다. (Windows 11은 설정 → 고급 네트워크 설정)
4네트워크 어댑터 속성 열기
사용 중인 Wi-Fi 또는 이더넷 어댑터를 우클릭 → 속성을 선택합니다.
5IPv4 설정 변경
목록에서 인터넷 프로토콜 버전 4 (TCP/IPv4)를 선택하고 속성 버튼을 클릭합니다.
6DNS 주소 입력
다음 DNS 서버 주소 사용을 선택 후 입력:

기본 설정 DNS 서버: 8.8.8.8
보조 DNS 서버: 8.8.4.4

입력 후 확인 클릭.
7브라우저 재시작 및 확인
브라우저를 완전히 닫고 다시 열어 여기여 사이트에 접속해 보세요. 만약 여전히 안 된다면 브라우저 캐시를 지우거나(Ctrl+Shift+Delete) 컴퓨터를 재시작합니다.

Mac에서 DNS 변경 방법

1시스템 환경설정 열기
화면 왼쪽 상단 Apple 로고 클릭 → 시스템 환경설정 (macOS Ventura 이상은 시스템 설정) 선택.
2네트워크 선택
네트워크를 클릭하고, 왼쪽 목록에서 현재 연결된 Wi-Fi 또는 이더넷을 선택합니다.
3고급 설정
고급... 버튼을 클릭한 후 상단 탭에서 DNS를 선택합니다.
4DNS 서버 추가
DNS 서버 목록 아래 + 버튼을 클릭하고 1.1.1.1을 입력합니다. 다시 +를 눌러 1.0.0.1도 추가합니다. 확인 → 적용을 클릭합니다.

Android에서 DNS 변경 방법

Android는 제조사마다 설정 메뉴 위치가 다를 수 있습니다. 아래는 대부분의 Android 기기에서 공통으로 적용되는 방법입니다.

1설정 → Wi-Fi
스마트폰 설정 앱을 열고 Wi-Fi 또는 연결 항목으로 이동합니다.
2현재 연결된 Wi-Fi 설정 열기
연결된 Wi-Fi 이름을 길게 누르거나, 네트워크 이름 옆의 ⚙️ 아이콘을 탭합니다.
3IP 설정을 정적으로 변경
IP 설정을 DHCP → 정적(Static)으로 변경합니다. IP 주소, 게이트웨이 등 기존 정보는 그대로 유지합니다.
4DNS 입력
DNS 1: 1.1.1.1
DNS 2: 1.0.0.1
입력 후 저장을 탭합니다.
⚠️ 안드로이드 9 이상에서는 설정 → 연결 → 기타 연결 설정 → 프라이빗 DNS에서 1dot1dot1dot1.cloudflare-dns.com을 입력하는 방법도 있습니다. 이 방법은 모든 인터넷 연결(Wi-Fi + 데이터)에 적용됩니다.

iPhone / iOS에서 DNS 변경 방법

1설정 → Wi-Fi
iPhone 설정 앱을 열고 Wi-Fi로 이동합니다.
2연결된 Wi-Fi 정보 열기
현재 연결된 Wi-Fi 이름 옆의 ⓘ 아이콘을 탭합니다.
3DNS 설정
아래로 스크롤하여 DNS 구성을 찾습니다. 자동 → 수동으로 변경합니다.
4DNS 서버 추가
기존 DNS 서버를 모두 삭제하고 서버 추가를 탭하여 8.8.8.8과 8.8.4.4를 입력합니다. 오른쪽 상단 저장을 탭합니다.

DNS 변경 후에도 접속이 안 될 때

DNS 변경 후에도 접속이 안 된다면 다음 추가 방법을 시도해 보세요.

  • 브라우저 DNS 캐시 삭제: Chrome에서 chrome://net-internals/#dns 접속 후 "Clear host cache" 버튼 클릭
  • Windows DNS 캐시 플러시: 명령 프롬프트(관리자)에서 ipconfig /flushdns 실행
  • VPN 사용: Proton VPN, Windscribe 등 무료 VPN 앱을 설치하고 해외 서버에 연결
  • 다른 브라우저 시도: Chrome, Firefox, Edge 등 다른 브라우저에서 접속 시도
  • 시크릿 모드: 브라우저 시크릿/프라이빗 모드에서 접속 시도
Windows DNS 캐시 삭제 명령어:
ipconfig /flushdns
