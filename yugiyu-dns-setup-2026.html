여기여 DNS 변경 방법 2026 — Windows · Mac · Android · iOS 완벽 가이드

여기여에 접속하기 위한 DNS 변경 방법을 모든 기기별로 총정리했습니다. Windows 10/11, Mac(macOS), Android 스마트폰, iPhone(iOS)에서 구글 DNS(8.8.8.8)와 클라우드플레어 DNS(1.1.1.1)로 변경하는 방법을 매우 상세하게 안내합니다.

운영자: 여기여 (Yugiyu) | 주소모음 포털 운영팀

DNS 변경 전 알아야 할 것

DNS(Domain Name System) 변경은 인터넷 주소 조회 경로를 바꾸는 작업입니다. 국내 ISP(통신사) 기본 DNS 대신 구글이나 클라우드플레어의 해외 DNS를 사용하면, 국내에서 차단된 사이트에도 접속할 수 있습니다. 설치 파일이 없고, 완전 무료이며, 속도 저하도 거의 없습니다.

구글 DNS — 기본: 8.8.8.8 / 보조: 8.8.4.4
클라우드플레어 — 기본: 1.1.1.1 / 보조: 1.0.0.1
🪟
Windows 10 / 11
제어판 또는 설정 앱 사용
  1. 설정 열기: Windows 키 + I → 네트워크 및 인터넷 클릭
  2. 어댑터 옵션 변경: 이더넷 또는 Wi-Fi 탭 → "어댑터 옵션 변경" (Win11: 고급 네트워크 설정)
  3. 속성 열기: 사용 중인 네트워크 어댑터 우클릭 → 속성
  4. IPv4 더블클릭: 목록에서 "인터넷 프로토콜 버전 4(TCP/IPv4)" 더블클릭
  5. DNS 입력: "다음 DNS 서버 주소 사용" 선택 → 기본: 8.8.8.8 / 보조: 8.8.4.4
  6. 확인 클릭: 모든 창에서 확인 클릭 후 브라우저 재시작
💡 Win11은 설정 → 네트워크 및 인터넷 → Wi-Fi/이더넷 → "DNS 서버 할당" 옆 편집 버튼으로도 변경 가능합니다.
🍎
macOS (Mac)
시스템 환경설정 사용
  1. 시스템 환경설정 열기: 화면 왼쪽 상단 Apple 로고 → 시스템 환경설정 (macOS Ventura+는 시스템 설정)
  2. 네트워크 선택: "네트워크" 클릭 → 왼쪽 목록에서 현재 연결된 Wi-Fi 또는 이더넷 선택
  3. 고급 버튼: 오른쪽 하단 "고급..." 버튼 클릭 → 상단 "DNS" 탭 선택
  4. DNS 추가: 왼쪽 목록 아래 "+" 버튼 클릭 → 1.1.1.1 입력. 다시 "+"→ 1.0.0.1 입력
  5. 저장: "확인" → "적용" 버튼 클릭
🤖
Android (안드로이드)
Wi-Fi 정적 IP 또는 프라이빗 DNS

방법 A — Wi-Fi 설정 (모든 Android)

  1. 설정 → Wi-Fi: 연결된 Wi-Fi 이름을 길게 누름 → "네트워크 수정" 또는 ⚙️ 아이콘 탭
  2. 고급 옵션 보기: "고급 옵션" 또는 "자세히" 펼치기
  3. IP 설정 변경: "IP 설정"을 DHCP → 정적으로 변경
  4. DNS 입력: DNS 1: 1.1.1.1 / DNS 2: 1.0.0.1
  5. 저장: 저장 버튼 탭 → Wi-Fi 끊었다 재연결

방법 B — 프라이빗 DNS (Android 9 이상 권장)

  1. 설정 → 연결 → 기타 연결 설정 → 프라이빗 DNS
  2. "프라이빗 DNS 공급자 호스트 이름" 선택
  3. 1dot1dot1dot1.cloudflare-dns.com 입력 후 저장
💡 프라이빗 DNS(방법 B)는 Wi-Fi와 모바일 데이터 모두에 적용되어 더 편리합니다.
📱
iPhone / iPad (iOS)
Wi-Fi 설정에서 DNS 수동 변경
  1. 설정 → Wi-Fi: iPhone 설정 앱 → Wi-Fi 탭
  2. 네트워크 정보: 현재 연결된 Wi-Fi 이름 오른쪽 ⓘ 아이콘 탭
  3. DNS 구성: 아래로 스크롤 → "DNS 구성" 탭 → 자동 → 수동으로 변경
  4. 기존 서버 삭제: 기존 DNS 서버 옆 빨간 − 버튼을 탭해 모두 삭제
  5. 새 DNS 추가: "서버 추가" 탭 → 8.8.8.8 입력. 다시 "서버 추가" → 8.8.4.4
  6. 저장: 오른쪽 상단 "저장" 탭

DNS 변경 확인 방법

DNS가 제대로 변경되었는지 확인하려면 브라우저 주소창에 https://1.1.1.1/help 를 입력하세요. "Using DNS over HTTPS (DoH)" 또는 현재 사용 중인 DNS 서버 정보가 표시됩니다.

또는 Windows 명령 프롬프트에서 nslookup google.com을 실행하면 현재 사용 중인 DNS 서버 주소가 표시됩니다.

DNS 원래대로 되돌리는 방법

여기여 이용이 끝나거나 원래 설정으로 되돌리고 싶다면, 위 단계를 역순으로 진행하여 DNS를 "자동 획득(DHCP)"으로 변경하거나, 입력했던 DNS 주소를 삭제하면 됩니다.
